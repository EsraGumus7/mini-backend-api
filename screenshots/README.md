# Screenshots

Bu klasöre Postman test ekran görüntülerini ekleyin.

## Önerilen Ekran Görüntüleri:

1. **health-check.png** - `GET /` endpoint testi
2. **register.png** - `POST /api/auth/register` endpoint testi
3. **login.png** - `POST /api/auth/login` endpoint testi
4. **get-all-items.png** - `GET /api/items` endpoint testi
5. **create-item.png** - `POST /api/items` endpoint testi (token ile)
6. **user-profile.png** - `GET /api/users/profile` endpoint testi (token ile)

## Nasıl Eklenir:

1. Postman'de test yaparken ekran görüntüsü alın (Windows: `Win + Shift + S`)
2. Dosyayı bu klasöre kaydedin
3. Dosya adını yukarıdaki isimlerle eşleştirin
4. README.md'deki görüntüler otomatik olarak görünecektir
